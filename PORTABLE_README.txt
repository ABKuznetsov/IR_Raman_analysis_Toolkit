IR/Raman Phase Finder 1.0 portable source package

This folder contains the application source, launch scripts, documentation,
examples, icons, and the reference-template file.

Not included:
- .git history
- local .venv
- test cache / Python cache
- downloaded spectral databases
- old build archives

macOS:
1. Install Python 3.11 or 3.12 if it is not installed.
2. Run setup_env.command once to create the local .venv.
3. Run run_finder.command to start the program.

Windows:
1. Run run_finder.bat.
2. If the shared Sci Python runtime is missing, the script will create/update it.

Reference databases are not distributed in this package.
Use the Databases tab in the program to download/index allowed public sources
or load user reference folders/templates.
